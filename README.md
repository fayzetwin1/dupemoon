данный проект отвечает на один простой вопрос: "а возможно ли из бездушного ИИ-ассистента сделать полноценную самостоятельную личность?"

## как это работает вкратце?

проект написан на python (aiogram) и общается с локальной нейронкой через ollama. 

главные фичи, которые делают его "живым":
1. **двойная генерация (окно мыслей).** бот не отвечает сразу. сначала он генерирует свои нефильтрованные мысли внутри тега `<thought>`, обдумывает твое сообщение, и только потом пишет публичный ответ. это дает моделям +100 к логике и эмпатии.
2. **абсолютно без цензуры.** используются abliterated модели (рекомендуется `Qwen2.5-7B-Instruct-Abliterated`). он может обсуждать любые темы, использовать сленг и не стесняться.
3. **эмоциональная инерция.** у бота есть свое "настроение", которое хранится в базе и плавно меняется в зависимости от диалога. если его разозлить, он не остынет за одно сообщение.
4. **фоновая жизнь.** под капотом крутится планировщик (`apscheduler`), который раз в 30 минут заставляет бота рефлексировать в фоне, даже если ты ему ничего не пишешь.
5. **фаза знакомства.** При первом запуске бот ниче не понимает, так как у него не сформирован характер. Он общается с тобой, собирает инфу, а потом генерирует свой собственный перманентный характер на основе вашего вайба.

## стек технологий

* **llm engine:** ollama (локальный инференс gguf моделей).
* **бэкенд:** python 3 + aiogram 3.
* **базы данных:** sqlite (для чатов, мыслей и настроения) + lancedb (векторная база для долговременной памяти фактов).
* **шедулер:** apscheduler (асинхронный).

## как запустить

1. установи зависимости: `pip install -r requirements.txt`
2. скопируй `.env.example` в `.env` и впиши туда свой токен от тг
3. скачай модель в Ollama: `ollama run hf.co/mradermacher/Qwen2.5-7B-Instruct-abliterated-GGUF:Q4_K_M`
4. запусти бота: `python main.py`

а и да, для работы этого чуда нужна видюха с более чем 4gb vram, иначе бот будет тупить и лагать (что очевидно)
